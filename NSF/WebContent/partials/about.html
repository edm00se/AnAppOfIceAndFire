An App of Ice and Fire

This is an application which demonstrates a segregated back-end and front-end. The back-end is a Domino/XPages server, with a front-end in AngularJS with Bootstrap 3 theme. The back-end can be mocked with json-server, as the GitHub Pages site for this app's repository shows.
